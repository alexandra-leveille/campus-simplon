Instance LifeCycle

- new Vue() =>
hook: beforeCreate()

instance Created =>
hook: created()

Compile template OR el's template =>
hook: beforeMount()

hook: mounted() Mounted to DOM <=> Data Changed
hook: beforeUpdate()
Re-Render DOM
hook: updated() (back to Mounted to DOM)

hook: beforeDestroy()

Destroyed => hook: detroyed()
